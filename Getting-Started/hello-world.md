# SYNTAX
1. using echo to print out statements  
`<?php
    echo "Hello, World!\n";
?>`  
**output will be Hello, World!**  

# alternatively, you can use print
`<?php
    print "Hello, World!\n";
?>`
**output will be Hello, World!**

# PHP C-style print-f 
`<?php
    printf("%\n", "Hello World!");
?>`
