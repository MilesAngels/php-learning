# SYNTAX
1. using echo to print out statements<br/>
`<?php
    echo "Hello, World!\n";
?>`<br/>
**output will be Hello, World!**<br/>

### alternatively, you can use print<br/>
`<?php
    print "Hello, World!\n";
?>`<br/>
**output will be Hello, World!**<br/>

### PHP C-style print-f <br/>
`<?php
    printf("%\n", "Hello World!");
?>`<br/><br/><br/>
